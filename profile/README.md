## Transfer Problems and Practical Fixes

- MetaMask bridge not working? Refresh the wallet connection, confirm the selected network is active, and check whether the route is currently supported.
- MetaMask bridge transaction time longer than expected? Review the transaction status, source-chain confirmation progress, and destination-chain congestion.
- MetaMask Bridge fees look too high? Wait for lower network activity, compare another route, or reduce urgency if the transfer is not time sensitive.
- MetaMask bridge minimum amount warning appears? Increase the transfer size or choose a different token and network pair if available.
- Bridge tokens MetaMask route unavailable? Recheck MetaMask bridge networks and confirm that the specific asset is supported on both chains.
